# Information for Copyright Holders

This application does not contain any information other than descriptions of individual files or
groups of files obtained from open sources on the Internet, which are held by certain users of
the Internet.

Proof that a user has these files may sometimes be a special set of values representing the
results of applying one-way hash functions to these files or their groups. These sets of values
are combined by users into a single whole and may be posted by them together with descriptions
of the files they have.

The developer of this application does not have the physical ability to verify that the file
descriptions posted by users, as well as the hash sets they create from files, correspond to
real files that users have, rather than being created in some other way.

In case of suspicions about the legality of certain files being on users' computers, you should
contact the users directly for confirmation or refutation of the suspicions. The developer does
not consider complaints that the descriptions and hashes provided by users may correspond to
files that the user does not have the right to store or distribute.

Exceptions are cases where the file description itself violates the legislation of the country
where the application is distributed or where the user resides. In this case, you can send a
reasoned complaint (with references to the legislation) to
[vasicdigital@mail.ru](mailto:vasicdigital@mail.ru).
All properly formatted requests will be reviewed within 72 hours.
