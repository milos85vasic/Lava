Decision 2 — Provider Label Rendering

How should each search result show which provider it came from?

Context

When searching across multiple providers, each result item needs to show its source. Current results have no provider indication at all. We need to add one that's clear but not distracting.

A

Colored Chip / Tag

A small colored badge next to the title with provider name

RuTracker Ubuntu 24.04 LTS Desktop ISO
RuTor Ubuntu Server 24.04 LTS
ArchiveOrg Ubuntu Linux Documentation
B

Subtle Icon + Color Bar

A small colored left border with the provider icon

Ubuntu 24.04 LTS Desktop ISO
Ubuntu Server 24.04 LTS
Ubuntu Linux Documentation
C

Compact Chip + Color (Recommended)

A tiny 2-3 letter chip with provider abbreviation, placed inline before the title — minimal footprint, maximum scanability

RT Ubuntu 24.04 LTS Desktop ISO
RI Ubuntu Server 24.04 LTS
IA Ubuntu Linux Documentation

Option C rationale (recommended)

  • Extremely compact — doesn't steal title space
  • Scannable at a glance — color + abbreviation
  • Works in dense result lists
  • Low visual weight, high information density

Trade-offs

  • Users need to learn abbreviations
  • New providers need 2-letter codes
  • Less playful than full-name chips
