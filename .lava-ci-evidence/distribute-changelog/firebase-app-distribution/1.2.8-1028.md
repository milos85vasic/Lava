# Lava 1.2.8 (1028)

## What's new

- **Multi-provider search** — search across all 6 providers (RuTracker, RuTor, NNM-Club, Kinozal, Internet Archive, Project Gutenberg)
- **Real-time streaming** — results appear as each provider responds, with provider labels on every card
- **New onboarding wizard** — 4-step setup: Welcome, pick your providers, configure each, and start exploring
- **Sync Now buttons** — tap to immediately sync favorites and bookmarks
- **Ocean, Forest, Sunset themes** — new color themes in Menu → Theme
- **Provider filter chips** — filter search results by provider
- **Redesigned Credentials screen** — modern look, fixed floating button overlap with nav bar
- **Crashlytics tracking** — better error reporting across all screens
